1. What is DOM and what is the purpose of DOM?

DOM means Document Object Model. It is a programming interface. by using this, we can create, change, or remove elements from the document.

2. How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML DOM elements by using five methods.

  1. 1. document.getElementsByTagName()
  2. 2. document.getElementsByClassName()
  3. 3. document.getElementById()
  4. 4. document.querySelector()
  5. 5. document.querySelectorAll(

3. What is event bubble?

Event bubbling is a way of event propagation in the HTML DOM. It relates to the order in which events are propagated in nested elements.

4. What is a callback function and why will you use it?

A callback is a function that is passed as an argument to another function. The callback function is called in the outer function to execute an action.
